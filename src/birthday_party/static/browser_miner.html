Birthday Party WebGPU Miner

Check the browser console (F12) for output.

Usage

Dry Run (testing)

http://localhost:8080/static/browser_miner.html

Submit to Server

http://localhost:8080/static/browser_miner.html?username=YOUR_USER&usertoken=YOUR_TOKEN

Custom Configuration

http://localhost:8080/static/browser_miner.html?username=USER&usertoken=TOKEN&server=http://localhost:8080&dp_bits=16

URL Parameters

  • username - Your username (required for server submission)
  • usertoken - Your user token (required for server submission)
  • server - Server URL (default: http://localhost:8080)
  • dp_bits - Distinguished point bits (default: 16)
  • dry_run - Set to "true" to disable submission
